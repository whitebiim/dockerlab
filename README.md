# Docker Lab CI/CD Project

![CI Status](https://github.com/whitebiim/dockerlab/workflows/CI%20Pipeline/badge.svg)

This project demonstrates CI/CD pipeline with GitHub Actions for Docker image building.

## Features
- Automated Docker image building
- Dockerfile linting with Hadolint
- Artifact storage for build logs and reports
- Quality gates for code quality

## CI/CD Pipeline
The pipeline includes:
1. **Lint** - Dockerfile quality check
2. **Build** - Docker image building
3. **Test** - Container functionality test
